Bitcoin module for ATS
======================

Provides an API around the Bitcoin RPC interface.

Install
-------

Clone in the ATS 'contrib' directory with the name 'bitcoin':

    cd $ATSHOME/contrib
    git clone git://github.com/doublec/ats-bitcoin bitcoin
    cd bitcoin
    make
